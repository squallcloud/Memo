## ちょっとしたこと

* フォルダ階層だけコピーしたい
```
xcopy /t /E コピー元 コピー先
```


* コントロールパネルのシステムを開く
```
Winキー + pause
```

* 管理者実行
```
powershell start-process 実行するバッチ -verb runas
```

* フォルダ共有化
  - 共有化
  ```
  net share 共有名=ドライブ:パス
  ```
  - 解除
  ```
  net share 共有名 \\コンピュータ名 /DELETE
  ```
